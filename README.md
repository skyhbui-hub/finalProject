## Getting Started

### 1. Frontend

Navigate to the project folder and run:

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`

### 2. Backend

Navigate to the backend folder and run:

```bash
cd backend
node server.js
```

The server will be available at `http://localhost:3001`

### Troubleshooting
```bash
nvm use 22
npm run dev
```
